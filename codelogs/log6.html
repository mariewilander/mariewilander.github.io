HTML Links

For insight on HTML links, I looked up information on W3 Schools, Code Academy, Geeks for Geeks, and Mozilla Developer Network.

Those sources can be found here:

  • W3 Schools
  • Code Academy
  • Geeks for Geeks
  • Mozilla Developer Network

Differences

One difference I noticed between the four references is that Code Academy included GIFs to explain syntax and how example code looks for the user. W3 Schools included a chapter summary at the end to reiterate what was mentioned on the page. Geeks for Geeks used videos to explain outputs of its example code. Both Mozilla Developer Network and W3 Schools offer example code for you to try out right on their website.

Features

A code reference page should offer examples that help a variety of users, such as pictures, example code, animations and videos. The code should be explained in a way that someone who’s new to coding can understand what’s being said. Pictures and GIFs help explain what the code should look like and what the result of that code looks on an HTML webpage. Videos help show what happens when you click on that link with the option of replaying it.

W3 Schools stands out the most as particularly a good resource. The “try it yourself” feature makes it stand out because it makes it easy to experiment with new code without having to go to a different website to practice.

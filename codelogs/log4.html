CSS Properties: Color

The color property sets the foreground color of the text of that element. You can define colors by using keywords such as red, yellow, blue, or transparent. You can use hexadecimal codes using either six digits or three digits. You can use RGB values, as well as hue, saturation, and lightness values.

CSS Properties: Background-color

The background-color property sets the background color of an element. It fills the entire area behind the element, including padding and border. You can set the color similarly to the color property by using named colors, hexadecimal codes, RGB values, or HSL values. Web designers use this property to increase readability, include contrast, and when layering text over an image.

CSS Properties: Border-color

The border-color property sets the color of the borders of an element. You can set different sides of the border to a specific color by alternating colors in the CSS code, such as border-color: red green;.

CSS can accept colors in different value styles. Some examples are:

Named:{color: pink;}

Hexadecimal{color: #FFC0CB;}

RGB:{color: rgb(255, 192, 203);}

HSL:{color: hsl(349 73% 96%);}
